# Goalify

Project Link: https://goalify-e2043.web.app/ and Demo Video Link: https://www.youtube.com/watch?v=a2RyIlMqSdM

## About the Project
Goalify is a web app designed to help UW students manage their goals more effectively. 
The app guides students by helping them break big goals into daily actionable tasks, 
stay organized, and build consistent habits. 

### Features:

- User authentication (sign up, log in, log out)
- Goal breakdown into daily tasks by AI assistant
- Add tasks to daily to-do list
- Mark tasks as complete/incomplete
- Add new goals with a description and due date
- Goals auto-categorized by due date into: Current Goals (due today or later) and Past Goals (due before today)
- Custom react calendar component for selecting dates
- AI assistant for goal breakdown and task suggestions


## Build With
- Frontend: React with Vite
- Routing: React Router
- Database: Firebase Firestore
- Authentication: Firebase Authentication
- Styling: CSS 


## Getting Started
1. Create the Repository on GitHub (git clone https://github.com/Pimnipa226/final-project-438.git
)
2. Create the Vite + React Project
3. Get API keys from Gemini API
3. Set up Firebase project
4. Start the development process using `npm run dev` to run the app locally
5. Use `npm run build` to create a production build
6. Deploy the app using Firebase Hosting or any other hosting service
